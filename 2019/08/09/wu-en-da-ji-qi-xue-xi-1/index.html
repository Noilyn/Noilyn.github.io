机器学习

a computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with experience E.

平方误差代价函数对大多数线性回归已经比较有用了

cost function \[ J(\theta_0,\theta_1) = \frac{1}{2m}\sum^m_{i=1}(h_{\theta}(x^{(i)})-y^{(i)})^2\\ minimize \quad J(\theta_0,\theta_1) \] 平方差函数（平方代价函数）

梯度下降

Gradient descent algorithm

repeat until convergence(simultaneous update) \[ \theta_j := \theta_j=\alpha\frac{\delta}{\delta \theta_j}J(\theta_0,\theta_1,...,\theta_m) \] \(\alpha\) 学习速率

\(\alpha\)不变时，接近局部最小值时，导数项会减小，因此步子会越来越小，所需不需要改变\(\alpha\).

只有方阵有逆矩阵，逆矩阵不存在的矩阵---奇异矩阵（退化矩阵）

cost funciton \[ J(\theta) = \frac{1}{2m}\sum^m_{i=1}(h_\theta(x^{(i)})-y^{(i)})^2 \] gradient descent(针对线性回归) \[ \theta_j:= \theta_j-\alpha \frac{\delta}{\delta \theta_j}J(\theta)\\ = \theta_j - \alpha \frac{1}{m}\sum^m_{i=1}(h_\theta(x^{(i)})-y^{(i)})x^{(i)} \]

加快梯度下降的一些方式

特征缩放（feature scaling）

用来避免梯度下降的迭代次数过多（消耗掉值的范围）

ge t every feature into approximately a \(-1\leq x_i\leq 1\) range (relatively small range would be enough)

均值归一化（mean normalization）**

replace \(x_i\) with \(x_i-\mu_i\) to make features have approximately zero mean

\(x_i = \frac{x_i-\mu_i}{s_i}\)

\(\mu_i\) averge value of sample \(x_i\), \(s_i\) standard deviation OR (max-min)

关于\(\alpha\)的设定

If \(\alpha\) is too small, slow convergence

IF \(\alpha\) is too large: \(J(\theta)\) is too large: may not decrease on every iteration.

通常看曲线图是否收敛而不是看\(\epsilon\)是否收敛到需要的值

Polynomial regression (多项式回归)

如此选择特征值一定要注意特征缩放

Normal Equation (正规方程)（不需要归一化）

用于线性回归求\(\theta\),(解析解法)

微积分解法 \[ J(\theta_0,\theta_1,...,\theta_m) = \frac{1}{2m}\sum^m_{i=1}(h_\theta(x^{(i)})-y^{(i)})^2\\ \frac{\delta}{\delta \theta_j} J(\theta) = ...=0 \] For every \(j\), then solve for \(\theta_0,\theta_1,...,\theta_n\)

\(\theta = (X^TX)^{-1}X^Ty\), 这样就能得到使得代价函数最小化的\(\theta\)

weekness： need to compute \((X^TX)^{-1}\) slow if \(n\) is very large.

When \(n\) is large, using gradient descent, or using normal equation.

正规方程的不可逆性

what if \(X^TX\) is non-invertible. octave 使用 pinv() 伪逆

\(m\leq n\)时用正则化解决

逻辑回归（logistics regression）

用于分类问题

Thereadshold classifier output \(h_\theta(x)\) at 0.5:

​ if \(h_\theta(x)\geq 0.5\), predict \("y=1"\)

​ if \(h_\theta(x)<0\), predict "y=0“

预测值在0，1之间的假设函数。 \[ h_\theta(x) = \theta^Tx\\ z = h_\theta(x) \\g(z) = \frac{1}{1+e^{-z}} \] g(z) is named as sigmoid function or logistic function

\(h_\theta(x)\) = estimated probability that \(y=1\) on input \(x\)

即 \(h_\theta(x) = P(y=1|x;\theta)\)

决策边界（Decision boundary）

是假设函数的一个属性，决定于其参数而不是数据集的参数

cost function

linear regression cost function \[ J(\theta) = \frac {1}{m} \sum^m_{i=1}\frac{1}{2}(h_\theta(x^{(i)})-y^{(i)})^2 \] We define that \(Cost(h_\theta(x^{(i)}),y^{(i)})=\frac{1}{2}(h_\theta(x^{(i)})-y^{(i)})^2\)

Logstic function cost function(不可用上面的线性回归cost function 否则会非凸) \[ Cost (h_\theta(x),y) = \begin{cases}-log(h_\theta(x))\quad if\quad y=1\\ -log(1-h_\theta(x)),if\quad y=0\end{cases} \]

\[ J(\theta) = \frac{1}{m}\sum^m_{i=1}Cost(h_\theta(x^{(i)}),y^{(i)}) \] Equivalently, \(Cost(h_\theta(x),y)=-ylog(h_\theta(x))-(1-y)log(1-h_\theta(x))\)

使用最大似然估计发，从统计数据总可以得出这个代价函数最好（而且有凸性）

find \(min_\theta J(\theta)\)
